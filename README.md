# Widget

A React application built with TypeScript and Vite.

## Requirements

- [Node.js](https://nodejs.org/) (version 18+ recommended)
- [pnpm](https://pnpm.io/) package manager

## Installation

1. Navigate to the project directory:
   ```bash
   cd c:\_personal\future\widget
   ```
   *(Or wherever the repository is located on your machine)*

2. Install the project dependencies using pnpm:
   ```bash
   pnpm install
   ```

## Available Scripts

In the project directory, you can run:

### `pnpm dev`
Runs the app in the development mode.
Open [http://localhost:5173](http://localhost:5173) (or the port provided in the terminal) to view it in the browser.
The page will reload if you make edits.

### `pnpm build`
Builds the app for production to the `dist` folder.
It correctly bundles React in production mode and optimizes the build for the best performance.

### `pnpm preview`
Locally preview the production build after running `pnpm build`.

### `pnpm lint`
Lints the codebase using ESLint.
